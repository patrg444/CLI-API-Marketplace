Enhance Image

API Key Required

Apply AI-powered enhancements to improve image quality. This endpoint uses advanced algorithms to denoise, sharpen, and adjust colors automatically.

https://api.api-direct.com/v1/image-processing /enhance
Rate Limit: 100 req/min Cost: 1 credit per request

Request Body application/json

Parameter Type Description
image_url required string URL of the image to enhance. Must be publicly accessible.
enhancements object Specific enhancement options to apply
denoise boolean Remove noise from the image. Default: true
sharpen number Sharpening intensity (0-2). Default: 1.0
brightness number Brightness adjustment (-1 to 1). Default: 0
contrast number Contrast adjustment (-1 to 1). Default: 0
format string Output format: jpeg, png, webp. Default: same as input
quality integer Output quality (1-100). Default: 85

Request Example 

{
  "image_url": "https://example.com/photo.jpg",
  "enhancements": {
    "denoise": true,
    "sharpen": 1.2,
    "brightness": 0.1,
    "contrast": 0.2
  },
  "format": "jpeg",
  "quality": 90
}

Response 200 OK

Field Type Description

data.url string URL to download the enhanced image
